# STASHR

This is stashr.

With stashr you can stash a file wherever you are.

And then get it back wherever you will be.

## Usage

Stash some files

```bash
stashr foo.txt bar.txt
```

Get them back

```bash
stashr
```

## Install

Not yet published, run

rustup default nightly
